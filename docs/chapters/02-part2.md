# 第二部分：投注

在第一部分中，我們討論了如何塑造工作。在這一部分，我們將討論如何決定要做什麼工作。

投注是我們選擇要在下一個週期中進行的項目的過程。與傳統的產品開發不同，我們不使用無盡的待辦事項列表或產品積壓。相反，我們在每個週期開始時做出明確的決定，選擇一組有限的項目進行投資。
